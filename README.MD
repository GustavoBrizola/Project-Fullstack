# Portfolio Hub

A fullstack portfolio hub site designed to showcase a collection of various projects in one centralized location.

## 🚀 Purpose
The purpose of this project is to serve as a professional hub where visitors can explore different projects, understand their functionality, and see the technologies used in each.

## 🛠️ Tech Stack

### Backend
- **Framework:** [Spring Boot](https://spring.io/projects/spring-boot)
- **Language:** Java

### Frontend
- **Framework:** [React](https://react.dev/)
- **Build Tool:** Vite
- **Language:** JavaScript

## 📁 Project Structure
- `backend/`: Contains the Spring Boot application logic, API endpoints, and database configurations.
- `frontend/`: Contains the React application, UI components, and API integration.

## ⚙️ Getting Started

### Backend
1. Navigate to the `backend` directory.
2. Run `./mvnw spring-boot:run`.

### Frontend
1. Navigate to the `frontend` directory.
2. Install dependencies: `npm install`.
3. Start the development server: `npm run dev`.
